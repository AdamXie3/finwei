# FinWei - UAE Financial Literacy App

FinWei is a web application designed to enhance financial literacy across the UAE. It provides users with tools and knowledge to make informed financial decisions.

## Features

- AI-powered budget calculator using the 50/30/20 rule
- Interactive financial literacy lessons
- UAE-specific financial education content
- Islamic finance principles
- Modern, responsive UI

## Installation

1. Clone the repository
2. Create a virtual environment:
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

4. Run the application:
```bash
python app.py
```

5. Visit `http://localhost:5000` in your web browser

## Technology Stack

- Flask (Python web framework)
- SQLAlchemy (Database ORM)
- TailwindCSS (Styling)
- Font Awesome (Icons)
